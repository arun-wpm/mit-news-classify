MIT News Classifier

Overview

The MIT News Classification model was developed as a part of a Machine Learning (Natural Language Project) project for Summer 2020. It is available as a downloadable package and helps to classify news articles to predefined categories.

How to Install

Just run the following command in your terminal:

pip install mit-news-classify

At a Glance

The package consists of several modules that can utilised to compute tags, embeddings, categories, etc.

from mitnewsclassify import tfidf

tfidf.gettags("This is a news article about Obama.")
>>> ['finances', 'politics and government', 'united states politics and government', 'united states international relations', 'elections', 'presidents and presidency (us)', 'presidential elections (us)', 'election issues', 'presidential election of 2008']
